# Notes App Visual Mockups

## 1. Login Page
### Description:
The login page is designed with a clean and modern look, featuring a simple input form for email and password. The background is a subtle gradient, providing an inviting atmosphere.

### Color Scheme:
- Background: #f5f7fa  
- Input Fields: #ffffff  
- Button: #3498db  
- Text: #2c3e50

### UI Components:
- Email Input Field  
- Password Input Field  
- Login Button  
- Forgot Password Link  

### Animation:
- Button hover effect with color change.

---

## 2. Dashboard Grid View
### Description:
The dashboard displays a grid of notes with options to filter and create new notes. Each note card features a title, short description, and an option to edit or delete.

### Color Scheme:
- Background: #ffffff  
- Note Cards: #ecf0f1  
- Title Text: #34495e  

### UI Components:
- Grid Layout  
- New Note Button  

### Animation:
- Smooth transitions when adding or removing notes.
